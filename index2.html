I. Почему код даёт именно такие результаты?

Дано: var a = 1, b = 1, c, d;

2. c = ++a; alert(c);
потому что используется префиксный инкремент, который выводит фактическое значение переменной 1+1=2

3. d = b++; alert(d);
потому что используется постфиксный инкремент, который выводит значение переменной без учета изменений

4. c = (2+ ++a); alert(c);
в данном случае переменная а измененная в п.2 содержит значение 2, действием в данной строке, переменной "а" прибавляется еще один (префиксный инкремент), таким образом переменная уже имеет значение 3...., следовательно 2+3=5

5. d = (2+ b++); alert(d);
в данном случае переменная b изменена посредством постфиксного инкремента (в стр.3) и имеет значени 2, следовательно 2+2=4

6. alert(a);
в данном случае в строках 2 и 4 использован префиксный инкемент, таким образом строкой 6 мы выводим резултат, дважды измененной переменной (а)

7. alert(b);
в данном конкретном случае, к исходному значению переменной b=1, дважды применен постфиксный инкремент, т.е. к значению b=1 необходимо прибавить +1,+1 - что в сумме дает 3.
